# Course Designer · 课程设计方法论

> 「课程设计最隐蔽的失败点不是内容不够好，而是目标、活动和评估三者各走各路。」

系统化的课程设计方法论 skill。从教学目标诊断、评估对齐审计、内容序列规划、教学活动选择到差异化与反馈设计，全流程输出教-学-评一致的课程方案。

## 安装

```bash
cp -r output/course-designer-skill ~/.claude/skills/course-designer/
```

## 触发方式

- 设计一门新课程或重构现有课程方案
- 需要对课程目标做 Bloom 层级诊断和重写
- 需要检查课程的教学目标、评估和活动是否对齐
- 需要为不同水平的学生设计差异化路径
- 学生规模大，需要高效的反馈系统设计
- 课程内容太多课时不够，需要取舍策略
- 新开课或换课后需要快速产出结构化方案

## 操作模型摘要

| # | 模型 | 一句话 |
|---|------|--------|
| H1 | Three-Gate Alignment Audit | ILO↔评估↔活动 三闸门，红绿灯标识哪里脱节 |
| H2 | Verb Unpacking | "理解"不够 → 追问证据 → 映射精确 Bloom 动词 |
| H3 | Activity Selection Matrix | 目标层级 + 学生水平 + 时间 → 推荐活动格式 |
| H4 | Core + Choice Decomposition | 目标拆为必达核心集 + 拓展选择集 |
| H5 | Three-Layer Feedback | 量规保覆盖 + 互评扩频次 + 教师深度轮转 |
| H6 | Breadth-vs-Depth Rule | L1-L2 自习/讲座，L3-L5 案例/讨论，桥接案例省时间 |
| H7 | Flipped Compliance Rule | 低容量 + 支架 + 计分进门测，缺一则失败 |
| H8 | Low-Resource UDL | 听-看-做三通道 + choice board，零成本差异化 |

## 约束

- 覆盖认知类学科。技能/艺术/体育类需调整
- 默认 20-40 人班级。>60 人大班部分策略需降级
- 不替代学科专业知识。方法论框架，内容需教师把关
- 差异化设计需要一次性投入（2-4 小时），后续摊薄
- 信息截止 2026 年 5 月
