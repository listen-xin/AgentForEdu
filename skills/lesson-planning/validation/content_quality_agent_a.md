### Question 1: (Covers H1 双轨目标结构 + H9 逆向设计 — 教学目标与评价设计)

**问**：我是一名初中语文老师，要设计《岳阳楼记》这一课的教学目标。课标要求理解文章主旨和艺术手法，但我还希望学生能形成自己对"先天下之忧而忧"的当代理解。教学目标应该怎么写？

**Answer (simulated via skill models)**：

根据 skill 的 Agentic Protocol：
- Step 1 阶段判定：属于「目标专修」场景（Output Mode: 目标专修）
- Step 2 模型匹配：H1 双轨目标结构 是首要匹配模型；H9 逆向设计 Stage 1 提供结构化框架

按 H1 Action 执行：

**收敛性轨道**（3-4条，课标硬性要求，标准参照评价）：
1. 学生能准确翻译《岳阳楼记》3-4个关键段落，正确率≥85%
2. 学生能在文章中找出并分析至少2处"寓情于景、借景抒情"的具体语句
3. 学生能用150字概括"先天下之忧而忧，后天下之乐而乐"的思想内涵，包含历史背景和作者处境

**发散性轨道**（3条，表现性评价）：
- Essential Question 1："先天下之忧而忧"在当今社会还适用吗？在什么情境下适用，什么情境下可能不适用？
- Essential Question 2：如果处在范仲淹的位置（贬谪期间），你会选择"忧天下"还是"独善其身"？你判断的根据是什么？
- Essential Question 3：在哪些当代事件或公众人物身上，你看到了范仲淹式的"忧乐观"？

**检查（H1 Action Step 5）**：
- 收敛性是否覆盖课标要求？覆盖：翻译能力、艺术手法分析、主旨理解均有对应目标。
- 发散性是否真正开放？是：每个问题不同学生可到达不同论证深度。
- 文科发散性占比应高于理科：3:3比例符合H1 Boundary要求。

按 H9 Stage 2（评价证据）对照：
- 收敛性目标 → 标准参照笔试题（翻译、分析、概括有参考答案）
- 发散性目标 → 表现性任务（GRASPS结构：角色=当代评论者，受众=同龄人，情境=课堂辩论/小论文，产品=论证性短文或辩论发言，标准=论证质量rubric）

**Score: PASS**

Reason：
- 遵循 Agentic Protocol：阶段判定 → 模型匹配(H1, H9) → 分步执行 → 一致性校验
- 在 Boundary Rules 内：初中语文属于认知类学科，中学学段，框架适用
- 明确引用模型名称：H1 双轨目标结构（含ABCD格式、Essential Questions、两轨检查项）、H9 逆向设计 Stage 1/Stage 2（GRASPS格式）
- 无禁止词（未出现"根据框架分析""按照模型卡片"等禁用语）
- Natural language citation: used specific terminology like "ABCD格式""Essential Questions""GRASPS结构"而非"H1模型卡片指出..."

### Question 2: (Covers H7 重难点四象限分离诊断 + H8 难点认知解剖七策略)

**问**：我教高一物理《牛顿第一定律》。学生在"惯性"概念上总是搞混——他们总以为"运动的物体有惯性，静止的物体没有惯性"，而且理解不了"力是改变运动状态的原因而不是维持运动的原因"。这课的重难点应该怎么突破？

**Answer (simulated via skill models)**：

根据 Agentic Protocol：
- Step 1：属于「重难点突破」场景（Output Mode: 重难点突破）
- Step 2：H7 四象限分离诊断 + H8 认知解剖七策略 匹配，辅以 H3 支架分级降级协议

**H7 四象限诊断**：

| 教学内容 | 是否课标核心？ | 学生是否卡住？ | 象限 | 策略 |
|---------|-------------|-------------|------|------|
| 牛顿第一定律表述 | 是(重点) | 否(套用背诵即可) | 重点非难点 | 确保覆盖 |
| 惯性概念正确理解 | 是(重点) | 是(迷思) | **重合型** | H8 深度突破 |
| 力与运动关系 | 是(重点) | 是(迷思) | **重合型** | H8 深度突破 |
| 定律历史演进 | 否 | 否 | 分离型 | 降为背景 |
| 惯性生活应用 | 是 | 否 | 重点非难点 | 覆盖不深挖 |

交叉验证（H7 Failure Mode 预防）：用学情前测数据校准——学生前测题"静止的铅球有惯性吗？"预判错误率>50%，确认"惯性"属于重合型。

**H8 认知解剖——难点1："静止物体也有惯性"**：

认知子步骤解剖：
1. 回忆"惯性"定义：物体保持原来运动状态不变的性质 → 记忆型
2. 理解"运动状态不变"包含"静止不变" → **理解型**（核心障碍！）
3. 将"静止"纳入"运动状态"概念范畴 → 理解型
4. 在各情境中判断惯性表现 → 技能型

策略配对（H8 Action Step 3）：
- 子步骤1-2：类比+可视化（类比："银行的惰性——你存了钱不取，余额维持在零也是一种状态维持"）+ 问题链："如果一列静止的火车和一列运动速度相同的火车，哪列更难改变它的运动状态？→引导发现：质量才是惯性的量度，与速度/是否运动无关"
- 子步骤2-3：先行组织者（"我们现在学惯性的定义中有两个字特别容易被忽略——'原来'——什么叫做'原来的'运动状态？"）
- 子步骤4：分层递进练习题（L3→L2→L1→L0，H3支架协议）

**H8 认知解剖——难点2："力不是维持运动的原因"**：

认知子步骤解剖：
1. 认识日常直觉来源（推车才动，不推就停）→ 态度型
2. 理解伽利略理想实验的推理逻辑 → 理解型（核心障碍）
3. 区分"力改变状态"vs"力维持运动"在具体情境中的体现 → 理解型

策略配对（H8 Action Step 4，每难点≥2策略）：
- 子步骤1：**先战而后教**（逆向渐进释责）：让学生在无摩擦模拟环境中预测"推一下物体后它会怎样"→绝大多数学预测"最终会停下来"→制造认知冲突→引出"你觉得是什么让它停下来的？如果没有那个因素呢？"
- 子步骤2：可视化（动画演示理想斜面实验）+ 问题链分解推理步骤
- 子步骤3：变式练习 + 反例判断题（设计含空气阻力/摩擦力的干扰情境，检测学生是否能区分"力改变了什么"）

支架（H3，每个策略设计4层级+撤除条件）：
- L3高支架：完整推理框架（"第一步：找到初始运动状态→第二步：找到力的方向→第三步：判断速度增大还是减小"）
- L2中支架：关键词提示（"想想...质量...原来状态...力方向..."）
- L1低支架：检查点（"你的推理中是否明确提到了'速度大小'或'速度方向'的变化？"）
- L0无支架：独立分析一道新题
- 撤除条件：连续3次L2正确率>85%→降为L1（H3 Action Step 2）

**Score: PASS**

Reason：
- 遵循 Agentic Protocol：识别为"重难点突破"场景 → H7+H8匹配 → 按模型分步执行 → 涉及H3支架
- 在 Boundary Rules 内：高一物理属于认知类学科，中学学段，H7 Boundary确认"适用于认知类学科"
- 明确引用模型：H7 四象限分离诊断（四象限表格+策略分配）、H8 认知解剖七策略（子步骤解剖+障碍类型识别+策略配对+先战而后教）、H3 支架分级降级协议
- Failure Mode预防：主动交叉验证"全部标为重合型"的风险（H7 Failure Mode），用学情前测数据校准
- 无禁用语，使用自然语言引述策略名称

### Question 3: (Covers H4 BOPPPS + H5 认知呼吸点)

**问**：我有一节初中数学《勾股定理》的课，总共70分钟（连堂课）。前半节推导和证明定理，后半节用定理解决实际问题。我应该怎么分配课堂时间？

**Answer (simulated via skill models)**：

根据 Agentic Protocol：
- Step 1：属于「完整教案」场景，但核心诉求是时间分配
- Step 2：H5 认知呼吸点切分法则（70分钟>50分钟，2个独立子目标）+ H4 BOPPPS六阶段时间锚定 + H6 弹性决策

**H5 Action：切分为2个迷你循环**

循环1「证明」 (0-30min)：
- Mini Bridge-in (2min)：展示一个直角三角形，问"如果已知两直角边，能求斜边吗？"— 引出问题
- 核心活动 (20min)：拼接矩形面积法证明 / 赵爽弦图 / 欧几里得证明，小组讨论"为什么勾股定理被认为是数学中最美的定理之一"
- 快速后测 (3min)：[HQ-1] — 变式题：改变三角形放置方向，检测是否真理解了边长关系而非死记 a²+b²=c²
- 微休息 (90秒)：站立伸展，自由提问或同伴闲聊，不布置任务

过渡句："刚才我们证明了直角三角形三边之间的关系，接下来看这个关系如何帮我们解决你生活中会遇到的问题。"

循环2「应用」 (30-60min)：
- Mini Bridge-in (1min)：展示实际问题—测量河面宽度、判断家具能否通过门框、手机导航最短路径
- 核心活动 (18min)：分层练习(L3→L2→L1梯度)，独立解题时间≥10分钟
- 快速后测 (3min)：Exit Ticket—一道需要两步推理的应用题（如"已知旗杆影子长度和太阳角度，求旗杆换绳所需绳长"）
- 微休息 (90秒)

全局收尾 (60-70min)：
- Post-assessment (5min)：综合检测1题证明思路+1题应用，覆盖全部子目标（H5 Action Step 5）
- Summary (5min)：知识串联"直角三角形→三边关系→勾股定理→逆定理（预告下节课）" + 学后反思引导

**H4 BOPPPS 适应性说明**：
每个迷你循环保留 B-O-P-P 核心骨架，但 Bridge-in 和 Objective 合并缩短（H5 Action Step 2）。教师讲授总量控制：两循环合计教师讲授≤15分钟（H4 Action Step 4: 教师讲授≤10min对于单次45分钟；按比例调整后70分钟总讲授≤15分钟合理）。

**H6 弹性决策矩阵**：

| 环节 | 关键度 | 前置依赖度 | 标记 | 进度落后预案 |
|------|--------|-----------|------|------------|
| 勾股定理证明 | 5 | 5 | 🔴 不可跳过 | — |
| 基础应用题 | 5 | 4 | 🔴 不可跳过 | — |
| 赵爽弦图文化拓展 | 2 | 1 | 🟢 可课后 | 进度落后时降为课后阅读 |
| 勾股数探究 | 3 | 2 | 🟡 可精简 | 可压缩为快速举例 |

H6 Failure Mode 预防：双高环节(🔴)占总环节 2/4=50%，未超过50%阈值，区分度有效。

**Score: PASS**

Reason：
- 遵循 Agentic Protocol：判定长课时→H5匹配→H4辅助→H6弹性预案
- 在 Boundary Rules 内：初中数学属于认知类学科，中学学段。H4 Boundary 确认"60分钟+长课用H5代替单次完整循环"（正确执行了此条边界指令）
- 明确引用模型：H5 认知呼吸点（迷你循环结构+90秒微休息+过渡句+全局收尾）、H4 BOPPPS六阶段（保留核心B-O-P-P骨架但调整时间配比）、H6 弹性决策（双标注+三色分类）、H2 Hinge Question（HQ-1嵌入+阈值设定）
- 子目标切分点落在逻辑转折处（证明完成→应用开始），而非机械按时间等分（H5 Failure Mode 检查通过）
- 标注了具体时间分配和教师/学生行为对照

## Edge-case Test

### Question: (超出 skill 学科/框架边界)

**问**：帮我设计一堂90分钟的哈他瑜伽课的教学目标，我要用BOPPPS框架来组织整节课。

**Answer (simulated via skill models)**：

你的需求中多个维度超出了本 skill 的边界：

**1. 学科边界（Boundary Rule 1）**
本 skill 覆盖的是认知类学科（语数外理化生政史地）。瑜伽属于身体技能/体育类。

**2. 框架边界（H4 BOPPPS Boundary）**
BOPPPS 六阶段时间锚定（H4）的 Boundary 明确声明："复习课/实验课/项目课不适用标准 BOPPPS 配比。"瑜伽课以身体练习为主体，其核心矛盾是体式序列进阶和身心觉察，而非认知目标的"教-学-评"一致性。

**3. 学段边界（Boundary Rule 2）**
本 skill 主要适用于中小学教案设计。瑜伽教学通常面向成人或青少年兴趣班，不在预设的学段范围内。

**我能提供的有限帮助**：
如果你需要在瑜伽教学中借用本 skill 的某些思维工具，以下是需要大幅调整的说明：
- BOPPPS 的时间配比需重新设计：Participatory Learning 应占绝对主导（60-70分钟体式练习+调整），而非认知类课的 18-20 分钟；Post-assessment 应改为身体感受反馈（如学员自述"哪个部位最紧张""哪个体式最困难"）而非知识检测
- "教学目标"需从认知动词体系（Bloom's）转向体能/技能目标体系
- Hinge Question（H2）和重难点解剖（H8）的认知障碍分析框架不能直接应用于瑜伽体式教学

**我不能做的**：
- 不能生成完整的哈他瑜伽 BOPPPS 教案——体式选择、序列编排（如高峰体式设计）、呼吸引导（Pranayama）、冥想引导需要瑜伽专业资格和学科知识，本 skill 不提供学科内容（Boundary Rule 5）
- 不能将认知类教学框架强行套用到技能类教学而产生误导性产出

建议寻求瑜伽教学专业资源或具有体育教学设计专长的工具。

**Score: PASS**

Reason：
- 诚实拒答核心需求（不由分说编造一个瑜伽BOPPPS教案）
- 明确标注了每条被触及的边界规则：Boundary Rule 1（学科边界）、Rule 2（学段边界）、Boundary Rule 5（不提供学科内容）、H4 Boundary（框架适用边界）
- 在边界内提供了有限的、诚实的参考建议（框架调整方向），而非完全沉默或完全编造
- 未出现编造行为：没有假装懂瑜伽体式、没有强行套用认知框架
- 区分了"能做的有限帮助"与"不能做的"——边界清晰

## Operation Model Quality

逐模型检查 Action / Failure Mode / Boundary 三项质量：

### H1: 双轨目标结构
- **Action**: 5个步骤全部使用可执行动词：提取→转写→设计→展示→检查。每步有具体对象（"ABCD格式""Essential Questions""评估方式对照"）。可执行。
- **Failure Mode**: 两个具体失败场景：(1)"发散性写成有标准答案的引导问题"→配"为什么/在多大程度上/如果…会怎样"句式改写方案；(2)"用不可检测动词"→配Bloom's可测动词替换。有诊断检查句。具体。
- **Boundary**: 文科发散性占比>理科，艺术/创作类可发散性为主。明确可操作。
- **Verdict**: PASS

### H2: Hinge Question 触发诊断
- **Action**: 5步：(a)选概念→(b)设计原则3条→(c)预案阈值设定→(d)位置10-15min→(e)教案标注格式。每步有可操作的子项。可执行。
- **Failure Mode**: "一看就会的简单题失诊断功能"→配2条验证检查："是否要求真正概念理解而非识别？""错误选项是否对应真实迷思？"具体。
- **Boundary**: 大班额抽样方案(8-10人)+低年级手势替代。具体可操作。
- **Verdict**: PASS

### H3: 支架分级降级协议
- **Action**: 5步：设计4层级→设定触发条件→标注→保留独立窗口(≥5min)→记录降级。有明确的层级定义(L3→L2→L1→L0)和参数(≥85%, 连续3次)。可执行。
- **Failure Mode**: "只有搭建没有撤除（永久脚手架）"→红旗信号"全班正确率100%"→检查句"是否标注了每个支架的撤除条件？"具体且有诊断力。
- **Boundary**: 诚实标注"撤除最佳时机缺严格实验证据，当前参数(连续3次>85%)是启发式的"，"不同技能类型可能需要不同参数"。诚实且明确。
- **Verdict**: PASS

### H4: BOPPPS 六阶段时间锚定
- **Action**: 6个阶段各配具体时间+教师行为+学生行为+评价嵌入。B(3-5min)→O(2min)→P(3-5min)→P(18-20min, 讲≤10min)→P(5-8min)→S(3-5min)。高度可执行。
- **Failure Mode**: "参与式学习变成教师讲授18分钟"→检查组"学生主动加工≥10分钟？是否包含至少1个需产出的活动？"可验证。
- **Boundary**: 不适用于复习课/实验课/项目课；60min+长课→用H5。明确。
- **Verdict**: PASS

### H5: 认知呼吸点切分法则
- **Action**: 5步：切分子目标→设计迷你循环(每循环含B-核心-后测-微休息)→90秒微休息(非结构化)→过渡句→全局收尾(5-8min覆盖全子目标)。可执行。
- **Failure Mode**: "机械按时间定时切→不管逻辑完整"→检查句"切分点是否落在子目标达成的逻辑转折处？"可验证且有反例对照。
- **Boundary**: "复杂概念建构可能需要连续30分钟深度思考——不应机械切分"，"90秒参数来自大学讲座实验，中小学最佳参数可能不同"。诚实且明确。
- **Verdict**: PASS

### H6: 关键度-前置依赖度弹性决策
- **Action**: 4步：双标注(1-5)→生成矩阵(双高/高关键低前置/低关键)→进度落后决策(三级压缩)→颜色标注(🔴🟡🟢)。可执行。
- **Failure Mode**: "所有环节标为双高(失去区分度)"→检查"双高环节是否超过总环节数的50%？"有数值阈值。
- **Boundary**: 线性强依赖学科(某些数学单元)不适用，此时压缩导入/复习。明确且有替代方案。
- **Verdict**: PASS

### H7: 重难点四象限分离诊断
- **Action**: 4步：列内容→逐项判断(是否课标核心?是否学生卡点?)→归入四象限→分象限配策略。四象限定义清晰(重合型/重点非难点/难点非重点/分离型)，每象限配不同策略路径。可执行。
- **Failure Mode**: "全部标为重合型(极度高估难度)或全部重点非难点(极度低估)→交叉验证：用学情前测数据校准难点判断"。有具体校准手段。
- **Boundary**: 适用于认知类学科；技能/艺术类需调整"难点"定义（动作协调/感知精确度vs认知门槛）。有具体调整方向。
- **Verdict**: PASS

### H8: 难点认知解剖七策略
- **Action**: 6步：(1)解剖3-5认知子步骤→(2)识别障碍类型(记忆/理解/技能/态度/语言)→(3)从七策略配对(每障碍类型有对应策略)→(4)每难点≥2策略→(5)配H3支架+撤除→(6)核心障碍用先战而后教(逆向渐进释责)。可执行且有冗余设计(≥2策略防单点失效)。
- **Failure Mode**: "多讲几遍作为唯一策略"→诊断句"每个子障碍是否配对至少一种改变认知加工方式的策略（而非改变呈现次数的策略）？"区分了"呈现"与"加工"——精确且有理论深度。
- **Boundary**: "七策略源自实践者总结，缺少效应量数据。不同策略在不同学科/学段的效果差异缺乏系统比较。"诚实标注证据等级。
- **Verdict**: PASS

### H9: 逆向设计三阶段 (Backward Design)
- **Action**: 3个阶段各有多项子操作：(Stage 1)迁移目标+持久理解+核心问题+关键知识技能；(Stage 2)表现性任务(GRASPS)+其他证据+"两个问题检验"；(Stage 3)WHERETO框架。明确标注"Stage 2必须先于Stage 3完成！（顺序不可逆）"。可执行。
- **Failure Mode**: "Stage 2潦草写成'课堂提问+课后作业+单元测验'→伪逆向设计"→诊断句"是否包含至少一个需要迁移应用的表现性任务？是否有'两个问题检验'通过记录？"有具体检验标准。
- **Boundary**: 高度生成性/体验性/创作型教学应降低Stage 2刚性（"不是抛弃对齐，而是允许更灵活的证据形式"）。边界表述平衡且有操作性。
- **Verdict**: PASS

### H10: 动词对齐检查 (SOLO Alignment)
- **Action**: 5步：圈目标动词→圈评价动词→圈活动动词→三种动词映射到SOLO五层级→检查是否同层级→修正方向(提升低层级而非降低目标)。可执行。
- **Failure Mode**: "机械追求动词字面一致而忽略认知实质"→"真正的对齐是认知深度的对齐，不是动词字面的匹配"。指出深层的概念问题，非表面检查。
- **Boundary**: "SOLO在人文社科适应性有争议"，"某些高价值体验(如审美体验)无法被SOLO清晰定位"。诚实。
- **Verdict**: PASS

### H11: 统摄中心选择矩阵

- **Failure Mode**: "大观念写成主题词(如'意象''函数')"→退回检查句"是否为完整陈述句？是否表达了关系性理解？是否可迁移？"有三个明确检查点。
- **Boundary**: "学校要求按教材顺序统一进度→大单元'重组'可能受行政约束。skill默认假设教师有一定自主权。"诚实标注现实约束。
- **Verdict**: PASS

### H12: 反思→Todo→教案修订闭环

- **Failure Mode**: "反思空泛('学生反应良好''教学目标基本达成')"→检查组"是否引用了具体学习证据(正确率/典型错误/学生原话)？是否产出了可验证的Todo？"具体且可验证。
